CATATAN PENTING
- Memilih file TIDAK langsung menyimpan data.
- Tombol Import aktif setelah Validasi File selesai tanpa error.
- Import materi membuat/memperbarui Topik berdasarkan slug.
- Re-import materi memperbarui materi yang cocok dan mengganti ulang tujuan/bagian materinya.
- Import soal memakai kode_soal sebagai kunci unik; re-import memperbarui soal yang sama.
- Soal otomatis terhubung ke topik, sehingga tampil sebagai latihan topikal pada menu Belajar siswa.
- Jika kode_kisi_kisi belum ada, soal tetap diimpor tanpa relasi kisi-kisi dan sistem memberi peringatan.
- Import QUESTION tidak otomatis membuat paket tryout. Gunakan template Mapping Tryout bila ingin membentuk paket ujian.
- Tidak ada perubahan schema Prisma dan tidak diperlukan migration baru.
